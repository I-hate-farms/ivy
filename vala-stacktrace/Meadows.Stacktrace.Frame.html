Description:

public class Frame

A part of a stacktrace.

This class represent on instance of a frame, ie a particular location in a binary (application or library) on the system called by the application

Note: frames from system libraries without code information available are not displayed by default. See Meadows.Stacktrace.Stacktrace.hide_installed_libraries for how to display them.
